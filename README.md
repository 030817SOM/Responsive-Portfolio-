Wanga Somhlaba Portfolio 🌐

A modern, responsive personal portfolio website showcasing the work, skills, and projects of Wanga Somhlaba, a Software Developer and Computer Science student from Bizana, Eastern Cape.

🔗 Live Demo: https://somhlabaportfolio.netlify.app/

🚀 Features
Hero section with profile photo and introduction
About Me section detailing background and journey
Services: UI/UX Design, Graphic Design, Frontend Development
Skills section with technical and professional skills
Latest Projects showcase with images and links
Contact form for collaboration or inquiries
Dark mode support
Smooth animations for a modern UX
🛠 Technologies
Frontend: React, TypeScript
Styling: Tailwind CSS, Tailwind Animate
UI & Components: Radix UI, Lucide React
State & Data: React Query
Build Tool: Vite


💻 Installation
Clone the repo and install dependencies:
git clone https://github.com/030817SOM/Responsive-Potfolio.git
cd portfolio
npm install

Run the development server:
npm run dev

Visit http://localhost:5173
in your browser.

🏗 Usage
Update your personal information in src/pages components (Hero, About, Services, Skills, Projects, Contact)
Add project images to src/assets and update project components
Customize colors, fonts, and layout in tailwind.config.ts and index.css

📁 Project Structure
src/
├─ assets/         # Images and media
├─ components/     # Reusable React components
├─ pages/          # Main sections (Hero, About, Services, Skills, Projects, Contact)
├─ index.css       # Tailwind + custom styles
├─ main.tsx        # Entry point
└─ tailwind.config.ts
